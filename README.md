# 📝 Todo App

A modern and responsive Todo Application built using HTML, CSS, and JavaScript.

This app allows users to manage daily tasks efficiently with live counts and local storage support.

---

## 🚀 Live Demo
👉 https://vnandan14.github.io/todo-app/

---

## ✨ Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 🔢 Live count (Total / Completed / Pending)
- 💾 Data persistence using Local Storage
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage

---

## 📂 Project Structure

```
todo-app/
│── index.html
│── style.css
│── script.js
│── assets/
```

---

## 🧠 How It Works

- Each task is assigned a unique ID
- Tasks are stored in browser local storage
- Counts update dynamically when tasks change
- Data remains even after page refresh
